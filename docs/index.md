## Vòng đời Phát triển

```text
INSTALL → PLAN → BUILD → TEST → DEBUG → FIX → OPTIMIZE → DEPLOY → OPERATE
```

| Giai đoạn | Agent | Mô tả |
|-----------|-------|-------|
| **INSTALL** | Installer | Cài đặt bench, site, apps |
| **PLAN** | Planner | Thiết kế kiến trúc, DocType |
| **BUILD** | Backend + Frontend | Lập trình Backend/Frontend |
| **TEST** | Commands/test | Kiểm thử tự động |
| **DEBUG** | Debugger | Phân tích lỗi (read-only) |
| **FIX** | Fixer | Sửa lỗi theo Fix Loop 6 bước |
| **OPTIMIZE** | Performance | Tối ưu hiệu năng |
| **DEPLOY** | Commands/bench | Triển khai production |
| **OPERATE** | Remote Ops | Vận hành từ xa qua REST API |
